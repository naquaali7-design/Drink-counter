☁ Sync Setup

Connected to Firestore — syncing across all devices

Open the app on another device, tap ☁, paste the same firebaseConfig and this sync code.

Uses Firebase Firestore — free tier, no credit card needed.

1
Go to console.firebase.google.com → Create a project (any name, disable Analytics to go faster).
2
In your project, click </> Web to add a web app → give it any name → click Register app. Copy the firebaseConfig object it shows you and paste it in the box below.
3
Back in the Firebase console, go to Build → Firestore Database → Create database → Start in test mode → pick any location → Enable.

Already synced on another device? Paste that device's sync code above instead of the auto-generated one.
