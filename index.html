☁ Cross-device Sync

Connected — syncing across all devices

Open this same URL on any device and your data will stay in sync automatically.

Sync uses a free Supabase database. Setup takes about 5 minutes.

1
Go to supabase.com → create a free account → create a new project (any name/password/region).
2
In your project, open the SQL Editor and run this query:
create table if not exists soda_entries ( id text primary key, date text not null, iso text not null, ts bigint not null ); alter table soda_entries enable row level security; create policy "public_rw" on soda_entries for all using (true) with check (true);
3
Go to Project Settings → API. Copy the Project URL and anon / public key, then paste them below.
