☁ Sync Setup

Connected to Firebase — syncing across all devices

Open the app URL on another device, tap ☁, and enter the same database URL + sync code.

Uses a free Firebase Realtime Database — no login required on your devices, just a sync code.

1
Go to console.firebase.google.com → Create a project (any name, disable Analytics to save time).
2
In your project: Build → Realtime Database → Create database → Start in test mode → Enable.
Copy the database URL — it looks like:
https://your-project-default-rtdb.firebaseio.com
3
Paste the URL below. Your sync code is auto-generated — copy it to any other device.

Already set up on another device? Paste that device's sync code here instead of using the auto-generated one.
